# Watering Can Worlds

A standalone browser game inspired by Minecraft-like movement and block terrain.

## Features

- WASD + mouse movement with run/jump controls.
- Playable character is a watering can with funny legs and three arms.
- Weird flower enemies chase and damage the player.
- Lure enemies into the waterfall to melt them and win.
- Five suns in the sky for a surreal multi-planet lighting setup.
- Random gravity anomalies:
  - Zero gravity bursts
  - Heavy gravity collapse
  - Ground-slide anomaly that pushes everything sideways

## Run locally

Because ES modules are loaded in the browser, run from a local web server:

```bash
python3 -m http.server 4173
```

Then open http://localhost:4173.
